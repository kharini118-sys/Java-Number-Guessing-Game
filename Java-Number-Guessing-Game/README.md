# Java Number Guessing Game

## Objective
Develop a Java console application where the user guesses a randomly generated number.

## Features
- Random number generation (1 to 100)
- User guess checking
- Too high / Too low feedback
- Limited attempts
- Multiple rounds
- Score tracking

## Technologies Used
- Java
- Random Class
- Scanner Class
- Loops and Conditional Statements

## Folder Structure

```text
Java-Number-Guessing-Game/
├── src/
│   └── NumberGuessingGame.java
└── README.md
```

## Compile Program

```bash
javac src/NumberGuessingGame.java
```

## Run Program

```bash
java -cp src NumberGuessingGame
```
